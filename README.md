# HR Employee Attrition Analysis Using SQL

## About This Project

I created this project to practice SQL on a real-world HR dataset and improve my data analysis skills.
The dataset contains information about employees such as department, salary, job role, job satisfaction, work-life balance, and attrition status. My goal was to analyze the data and understand the factors that may contribute to employee attrition.
This project helped me gain hands-on experience in writing SQL queries and extracting business insights from data.

## Tools Used

* MySQL
* MySQL Workbench
* GitHub

## Dataset

The dataset contains information about **1,470 employees**, including:
* Age
* Gender
* Department
* Job Role
* Monthly Income
* Education
* Marital Status
* Job Satisfaction
* Work-Life Balance
* Attrition Status

## Questions I Explored

Some of the questions I answered using SQL
* How many employees work in the company?
* How many employees left the company?
* What is the overall attrition rate?
* Which department has the highest attrition?
* Which job roles have the highest salaries?
* How is the workforce distributed by gender and education?
* How satisfied are employees with their jobs and work environment?

## SQL Concepts Used

Throughout this project, I practiced:
* SELECT
* WHERE
* GROUP BY
* ORDER BY
* Aggregate Functions (COUNT, SUM, AVG)
* CASE Statements
* Subqueries
* Window Functions
* RANK()
* ROW_NUMBER()

## Key Insights

* The overall employee attrition rate is approximately **16.12%**.
* The Sales department has the highest attrition rate.
* Research & Development has the largest number of employees.
* Managers earn the highest average salary.
* Employees with lower satisfaction scores are more likely to leave the company.
* Work-life balance appears to have an impact on employee retention.
